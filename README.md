### How to run

- Clone this repo running `git clone https://github.com/99ltaylor/acebook-all-css-no-style.git`

- Run these commands
```
bundle install
bin/rails db:create
bin/rails db:migrate
bundle exec rspec
```

- Run `rackup` or bin/rails server to start the web server
